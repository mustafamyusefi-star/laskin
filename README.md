Advanced Python Calculator
Description

A feature-rich Python calculator that supports basic arithmetic, advanced math functions (square root, factorial, exponent, logarithm), and trigonometry (sin, cos, tan). Keeps a history of calculations for easy reference.

Features

Addition, Subtraction, Multiplication, Division

Exponentiation and Square Root

Factorial and Logarithms

Sine, Cosine, and Tangent

Calculation History

How to Use

Clone the repository.

Run python calculator.py in your terminal.

Follow the interactive menu to perform calculations.

Requirements

Python 3.x

No external libraries required
